什麼是 PyGMT？

PyGMT 是基於 GMT（Generic Mapping Tools）的 Python 介面，讓你能以 Python 語法製作出版級的地圖與地理資料視覺化。從地震分佈、測深資料、等高線到全球海陸圖，都能輕鬆實現！

如果你喜歡 Python，又需要強大的地理視覺化工具，PyGMT 絕對是你的好夥伴。

PyGMT 的特色

  • 強大而精準的地圖繪製能力
  • 完整整合 GMT 的功能
  • 可用於學術研究與出版級圖件
  • 支援多種投影、色階與地理資料套件

範例：生成世界地圖

import pygmt
fig = pygmt.Figure()
fig.coast(region="d", projection="W0/6i", land="lightgray", water="skyblue")
fig.show()

以下示例展示如何使用 grdimage 以 geo 色階渲染全球地形資料：

fig = pygmt.Figure()
